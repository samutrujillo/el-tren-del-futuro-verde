# Archivos de Audio Requeridos

Para que el sonido funcione correctamente en "El Tren del Futuro Verde", por favor sube los siguientes archivos `.mp3` a esta carpeta (`/public/sounds/`).

## Efectos de Sonido (Globales)
- `pop.mp3` (Al agarrar un objeto)
- `success.mp3` (Acción correcta)
- `error.mp3` (Acción incorrecta)
- `win.mp3` (Nivel completado)
- `splash.mp3` (Objeto cae al agua - Estación 2)
- `click.mp3` (Click en interruptor - Estación 3)
